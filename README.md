# Obsidian AI/ML PhD Vault Template

An Obsidian vault template for AI/ML PhD students and researchers. It is organized around a practical research loop:

```text
papers -> ideas -> hypotheses -> experiments -> results -> manuscripts
```

## Structure

```text
00inbox/
01 Concepts/
02literature/
03meetings/
04 Experiments/
  01questions_hypotheses/
  02datasets/
  03model_designs/
  04training_runs/
  05evaluations/
  06ablations/
  07error_analysis/
  08reproducibility/
05presentations/
06conferences_events/
07training/
08admin_funding/
09student_supervision/
10ideas/
20 Research Log/
30manuscripts/
40projects/
50reading_lists/
60code_systems/
88 Templates/
  vault_templates/
99attachments/
_.vault_guide_and_constants/
```

## Templates

The active templates live in `88 Templates/vault_templates`:

- daily research log
- research project
- paper note
- advisor meeting
- group meeting
- research idea
- hypothesis
- dataset
- model architecture
- experiment plan
- training run
- evaluation
- ablation study
- error analysis
- benchmark result
- codebase note
- conference note
- manuscript outline
- review response
- reading list

## Obsidian Setup

The vault includes Obsidian settings for:

- Templates folder: `88 Templates/vault_templates`
- Daily notes folder: `20 Research Log`
- Daily note template: `88 Templates/vault_templates/00_.daily_research_log`

Open this folder as an Obsidian vault, then use the Templates core plugin to insert templates.
