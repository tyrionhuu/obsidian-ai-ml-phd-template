# Obsidian AI/ML PhD Vault Template

An Obsidian vault template for AI/ML PhD students and researchers. It is organized around a practical research loop:

```text
papers -> ideas -> hypotheses -> experiments -> results -> manuscripts
```

## Structure

```text
00 Inbox/
01 Concepts/
02 Literature/
  01 Reading Lists/
03 Ideas/
04 Projects/
  01 Side Projects/
  02 Collaborations/
  03 Proposals and Writing/
05 Experiments/
  01 Questions and Hypotheses/
  02 Datasets/
  03 Model Designs/
  04 Training Runs/
  05 Evaluations/
  06 Ablations/
  07 Error Analysis/
  08 Reproducibility/
  09 Code and Systems/
06 Results/

```

## Obsidian Setup

The vault includes Obsidian settings for:

- Templates folder: `88 Templates/Vault Templates`
- Daily notes folder: `20 Research Log`
- Daily note template: [00_.daily_research_log.md](<88 Templates/Vault Templates/00_.daily_research_log.md>)

Open this folder as an Obsidian vault, then use the Templates core plugin to insert templates.
