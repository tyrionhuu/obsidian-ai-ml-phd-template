# Obsidian AI/ML PhD Vault Template

An Obsidian vault template for AI/ML PhD students and researchers. It is organized around a practical research loop:

```text
papers -> ideas -> hypotheses -> experiments -> results -> manuscripts
```

## Structure

```text
00 Inbox/
01 Concepts/
02 Literature/
03 Meetings/
04 Experiments/
  01 Questions and Hypotheses/
  02 Datasets/
  03 Model Designs/
  04 Training Runs/
  05 Evaluations/
  06 Ablations/
  07 Error Analysis/
  08 Reproducibility/
05 Presentations/
06 Conferences and Events/
07 Training/
08 Admin and Funding/
09 Student Supervision/
10 Ideas/
100 People/
20 Research Log/
30 Manuscripts/
  01 Thesis/
  02 Papers/

88 Templates/

_ Vault Guide and Constants/
```

## Obsidian Setup

The vault includes Obsidian settings for:

- Templates folder: `88 Templates/Vault Templates`
- Daily notes folder: `20 Research Log`
- Daily note template: `88 Templates/Vault Templates/00_.daily_research_log`

Open this folder as an Obsidian vault, then use the Templates core plugin to insert templates.
